# Welcome to Unimind Content

We're excited to launch our new content hub built on MDX and Keystatic. This platform powers our blog, wiki, handbook, and custom landing pages — all stored as files in git.

## What's New

- **Git-backed content** — Every piece of content is a file committed to the repository
- **Notion-like editor** — Use Keystatic Studio to edit content without touching code
- **AI drafting** — Generate structured MDX content with our AI writer at `/adminx/ai`
- **Preview deployments** — Every branch gets a preview URL on Vercel

## Getting Started

Visit `/adminx` to open the Keystatic Studio editor.

> **Note:** Drafts are visible on preview deployments but hidden on production until published.
