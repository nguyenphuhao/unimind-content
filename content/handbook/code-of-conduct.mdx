# Code of Conduct

Our community is built on respect, collaboration, and a shared commitment to improving health outcomes.

## Our Values

- **Empathy** — We listen first and respond with care
- **Transparency** — We share information openly and honestly
- **Excellence** — We hold ourselves to high standards

## Expected Behavior

- Treat all community members with respect
- Provide constructive feedback
- Support each other's growth

<Callout type="warning">
Violations of this code of conduct may result in account suspension.
</Callout>
