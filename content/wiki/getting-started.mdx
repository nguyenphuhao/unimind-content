# Getting Started with Unimind

This wiki article covers the basics of getting started with the Unimind platform.

## Prerequisites

- A modern web browser
- An account invitation from the Unimind team

## First Steps

1. Accept your email invitation
2. Set up your profile
3. Explore the dashboard

> **Success:** You're all set! Welcome to the Unimind community.
